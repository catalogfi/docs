---
title: "Get User by Custom Auth"
openapi: "POST /v1/users/get-by-custom-auth"
---

This endpoint allows you to retrieve a user by their custom authentication ID. It's useful when you have your own authentication system and need to map your users to Catalog users.

## Request

The request requires the custom authentication ID in the request body.

## Response

Returns the user data including their ID, creation timestamp, and linked accounts if the user is found.

## Example

```bash
curl -X POST https://api.catalog.fi/v1/users/get-by-custom-auth \
  -H "Content-Type: application/json" \
  -H "privy-app-id: YOUR_APP_ID" \
  -d '{
    "custom_auth_id": "custom_user_123"
  }'
```

## Use Cases

- Integration with existing authentication systems
- Custom user ID mapping
- Migration from other platforms
- Enterprise SSO integrations
