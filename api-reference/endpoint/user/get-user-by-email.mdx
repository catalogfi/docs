---
title: "Get User by Email"
openapi: "POST /v1/users/get-by-email"
---

This endpoint allows you to retrieve a user by their email address. It's useful when you need to find a specific user using their email as an identifier.

## Request

The request requires the user's email address in the request body.

## Response

Returns the user data including their ID, creation timestamp, and linked accounts if the user is found.

## Example

```bash
curl -X POST https://api.catalog.fi/v1/users/get-by-email \
  -H "Content-Type: application/json" \
  -H "privy-app-id: YOUR_APP_ID" \
  -d '{
    "email": "user@example.com"
  }'
```

## Use Cases

- User lookup during authentication flows
- Account recovery processes
- User management in admin panels
- Integration with existing email-based systems
