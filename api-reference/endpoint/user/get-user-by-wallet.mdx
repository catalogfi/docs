---
title: "Get User by Wallet"
openapi: "POST /v1/users/get-by-wallet"
---

This endpoint allows you to retrieve a user by their wallet address. It's useful when you need to find a specific user using their wallet as an identifier.

## Request

The request requires the user's wallet address in the request body.

## Response

Returns the user data including their ID, creation timestamp, and linked accounts if the user is found.

## Example

```bash
curl -X POST https://api.catalog.fi/v1/users/get-by-wallet \
  -H "Content-Type: application/json" \
  -H "privy-app-id: YOUR_APP_ID" \
  -d '{
    "wallet_address": "0xE6bFb4137F3A8C069F98cc775f324A84FE45FdFF"
  }'
```

## Use Cases

- Wallet-based user lookup
- DeFi application integrations
- NFT ownership verification
- Cross-platform wallet linking
