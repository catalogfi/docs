# Introduction

Catalog Accounts underpin the way users interact with Web3 through Catalog. These are a set of keypairs and smart contracts that allow users to store assets from any blockchain, including Bitcoin, ERC-20 tokens, and NFTs. A Catalog Account consists of two wallets:

1. **Master Wallet**: The primary deposit address and the owner of the Instant Wallet.
2. **Instant Wallet**: A 2-of-2 multi-sig to bypass confirmations.
