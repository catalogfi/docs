# Introduction

Catalog Accounts underpin the way users interact with Web3 through Catalog. These allow users to store assets from any blockchain, and send them without waiting for confirmation delays. Catalog Accounts consist of a set of keypairs and smart contracts that allow users to store assets from any blockchain, including Bitcoin, ERC-20 tokens, and NFTs. These accounts consists of two wallets:

1. **Master Wallet**: The primary deposit address and the owner of the Instant Wallet.
2. **Instant Wallet**: A 2-of-2 multi-sig wallet that allows for instant transactions.
