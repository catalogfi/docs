# Gas payment abstraction

Catalog Accounts make use of the ERC-4337 standard, which allows them to leverage powerful account abstraction capabilities. The ERC-4337 standard enables a decentralized fee market, reducing transaction costs for account operations.

## Catalog Relayer

The Catalog Relayer is a service that participate in this market, setting flexible fee requirements affording users the freedom to pay fees in any asset within their Catalog Account.
