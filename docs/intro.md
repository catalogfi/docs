# Introduction

## What is Catalog?

[Catalog](https://catalog.fi/) is the first peer-to-peer protocol designed to allow Bitcoin, and any other asset, to seamlessly interact with the decentralized web. The Catalog protocol enables trustless and atomic cross-chain swaps without introducing a central custodian. By removing the requirement for synthetic assets, Catalog enables a single source of liquidity, eliminating the need to incentivize countless liquidity pools. Catalog introduces a novel non-custodial account system that facilitates instant cross-chain swaps. These accounts can store assets from any blockchain, and allows for them to be used without waiting for confirmation delays.
